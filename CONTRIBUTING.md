# Contributing

Thanks for your interest in contributing! This guide helps you get set up and
submit high-quality changes.

## Getting started

1. Fork the repository and create a branch from `main`.
2. Install dependencies: `npm install`
3. Copy `env.example` to `.env.local` and set `GEMINI_API_KEY`.
4. Start the dev server: `npm run dev`

## Development workflow

- Keep changes focused and small when possible.
- Follow the existing code style (TypeScript + React patterns).
- Prefer descriptive commit messages.

## Pull requests

Before opening a PR, please ensure:

- The app builds locally: `npm run build`
- New or updated behavior is documented in `README.md` when needed
- The PR description explains the problem and the solution

## Reporting issues

If you find a bug or want a new feature, open an issue using the templates.
